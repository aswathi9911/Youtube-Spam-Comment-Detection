# Explainable Deep Learning for YouTube Spam Comment Detection

## Overview
This project proposes a hybrid deep learning architecture combining DistilBERT, CNN, and BiLSTM for detecting spam comments on YouTube.

## Features
- DistilBERT contextual embeddings
- CNN feature extraction
- BiLSTM sequence learning
- Spam/Ham classification
- Explainable AI support

## Technologies
- Python
- TensorFlow
- Transformers
- Scikit-learn
- Pandas
- NumPy

## Dataset
YouTube Spam Collection Dataset

## Results
Achieved high classification performance for spam detection.
